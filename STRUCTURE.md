# Folders structure
```js
+-- app // App code
|   +-- Helpers // App helpers
|   |   +-- Classes // helpers class
|   |   +-- Functions // helpers functions
|   |   +-- * // helpers files
+-- Interfaces // App Interfaces
+-- Traits // App Traits
+-- lang // App translations
|   +-- * // Locale dir
|   |   +-- models // models translation dir
|   |   |   +-- * // model translation files
|   |   +-- * // other translation files
+-- Packages // custom packages
```
